# 🌦️ Weather App India

A modern weather application built with HTML, CSS, and JavaScript using WeatherAPI.

## Features

* Real-time weather data
* Search weather by city
* Temperature & feels-like temperature
* Humidity, Wind, Pressure
* Visibility & Cloud Coverage
* UV Index
* Dark / Light Theme Toggle
* Responsive Design

## Tech Stack

* HTML
* CSS
* JavaScript
* WeatherAPI
* Font Awesome
